## ⚙️ Características

- 🔍 Interfaz tipo Google / navegador
- ⌨ Barra de búsqueda funcional (simulada)
- ⏳ Animación de carga con puntos
- ⚠️ Pantalla de error falsa tipo servidor
- 🔗 Redirección a enlace externo
- 🎨 Estilo retro años 90

---

## 🚀 Cómo ejecutarlo

### 1. Requisitos
- Windows
- PowerShell 5 o superior

### 2. Ejecutar el script

```powershell
& "C:\Users\HP\Downloads\powershell.ps1"
````

## 🎮 Cómo usarlo

1. Abre el script en PowerShell
2. Escribe algo en la barra de búsqueda
3. Presiona **Search**
4. Espera la animación de carga
5. Verás la pantalla de resultados simulada
6. Puedes usar el enlace externo si aparece
